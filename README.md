# 🚀 AI Powered AWS Cloud Management Platform

A full-stack AI-powered AWS cloud management dashboard built using React, FastAPI, and AWS SDK (Boto3).  
This platform provides real-time AWS infrastructure monitoring, EC2 instance management, S3 bucket tracking, CloudWatch monitoring, JWT authentication, and an AI assistant for AWS operations.

## 🔐 Authentication

- Protected Routes
- Secure Password Hashing
